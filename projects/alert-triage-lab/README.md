# Alert Triage Lab

## Overview

This project documents SOC-style alert triage practice using simulated alerts and written analysis.

## Objectives

- Review alert details and identify relevant evidence
- Classify alert severity
- Decide whether escalation is needed
- Write concise triage notes

## Lab Environment

- Operating systems: Linux and Windows lab systems
- Tools used: Logs, sample alerts, documentation templates
- Lab type: Simulated SOC alert triage

## Tools Used

- Log review tools
- Markdown documentation
- Security report templates
