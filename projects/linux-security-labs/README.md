# Linux Security Labs

## Overview

This project documents Linux security practice focused on command-line usage, permissions, users, groups, logs, processes, and basic system review.

## Objectives

- Practice Linux security commands
- Review users, groups, permissions, and running processes
- Explore logs and basic system information
- Document commands, outputs, and lessons learned

## Lab Environment

- Operating systems: Parrot OS and Linux Mint
- Tools used: Linux terminal and built-in command-line utilities
- Lab type: Local Linux practice

## Tools Used

- Terminal
- `ls`, `chmod`, `chown`, `ps`, `top`, `ip`, `ss`, `journalctl`
- Markdown documentation
