# Network Traffic Analysis

## Overview

This project documents network traffic analysis practice using Wireshark and tcpdump in lab environments.

## Objectives

- Capture and inspect basic network traffic
- Identify common protocols such as DNS, HTTP, TCP, and ICMP
- Practice filtering traffic in Wireshark and tcpdump
- Document observations and findings clearly

## Lab Environment

- Operating systems: Parrot OS, Linux Mint, Windows 11
- Tools used: Wireshark, tcpdump
- Lab type: Local and controlled traffic analysis

## Tools Used

- Wireshark
- tcpdump
- Linux terminal
- Markdown documentation
