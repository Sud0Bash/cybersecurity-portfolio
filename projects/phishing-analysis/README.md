# Phishing Analysis

## Overview

This project documents phishing analysis practice using sample emails, suspicious links, file reputation checks, and written investigation notes.

## Objectives

- Identify common phishing indicators
- Review email headers and message content
- Check suspicious URLs or file hashes using tools such as VirusTotal
- Document findings in a clear, professional format

## Lab Environment

- Operating systems: Windows 11 and Linux lab systems
- Tools used: VirusTotal, browser-based analysis tools, text editor
- Lab type: Sample or sanitized phishing analysis

## Tools Used

- VirusTotal
- Email header review tools
- Browser developer tools or safe preview methods
- Markdown documentation
