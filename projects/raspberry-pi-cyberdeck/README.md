# Raspberry Pi Cyberdeck

## Overview

This project documents an ongoing Raspberry Pi 5 cyberdeck build used as a portable Linux-based cybersecurity learning platform.

## Objectives

- Configure a Raspberry Pi 5 with a touchscreen and Linux environment
- Build or improve a Python launcher interface
- Test hardware modules such as IR and NFC components
- Explore compatible wireless and SDR tools in a controlled, legal lab setting
- Document setup steps, troubleshooting, and lessons learned

## Lab Environment

- Hardware: Raspberry Pi 5, touchscreen, cyberdeck components
- Operating systems: Parrot OS or other Linux setup as tested
- Tools used: Python, Linux terminal, hardware modules
- Lab type: Personal hardware and Linux learning project

## Tools and Components

- Raspberry Pi 5
- Touchscreen
- Python launcher
- IR receiver
- NFC module
- Wi-Fi adapter with monitor mode support
- Planned SDR tools such as HackRF or RTL-SDR
